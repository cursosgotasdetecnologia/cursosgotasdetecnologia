## 🚀 Projeto em Destaque

### 🎭 [app-northwind — Automação E2E com Playwright](https://github.com/cursosgotasdetecnologia/app-northwind)

> Projeto profissional desenvolvido durante a formação Gotas de Tecnologia.
> Pipeline CI/CD configurado e rodando no GitHub Actions.

<!-- Caminho: GitHub > Actions > clique no run verde > tire screenshot > salve como pipeline.png -->
[![Playwright Tests](https://github.com/cursosgotasdetecnologia/app-northwind/actions/workflows/playwright.yml/badge.svg)](https://github.com/cursosgotasdetecnologia/app-northwind/actions/workflows/playwright.yml)

**O que foi implementado:**

| Habilidade | Detalhe |
|---|---|
| 🏗️ Page Object Model | Arquitetura escalável e reutilizável |
| 🔄 CRUD Automatizado | Cadastro, edição, exclusão e detalhes |
| 📊 Data Driven Testing | JSON e CSV com massa gerada por IA |
| 🎲 Faker.js | Dados dinâmicos a cada execução |
| 🔍 Validação de Combos | Fornecedores e categorias — existência e contagem |
| ⚡ Esperas Inteligentes | waitFor, toHaveCount, waitForURL |
| 📸 Evidências Automáticas | Screenshots e vídeos em falhas |
| 🕵️ Trace Viewer | Timeline completa de cada teste |
| 📋 Allure Report | Relatórios com severity, epic e feature |
| 🚀 CI/CD | Pipeline automático no GitHub Actions |
